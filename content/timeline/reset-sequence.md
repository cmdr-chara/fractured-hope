# Reset Sequence

The reset sequence covers how timeline rollback functions in _Fractured Hope_, when Chara uses it, and how the story removes that option permanently. This page covers mechanics and event sequence. For the full story order, see [[timeline/chronology|Chronology]].

## How Resets Work

Resets allow a person with sufficient [[lore/determination|Determination]] to roll back to a prior save state. Determination is the structural property that makes memory retention across a reset possible — without it, the reset would erase the person's continuity of experience along with the events themselves.

The story treats resets as a fallback, not a solution. Rolling back preserves the underlying damage, grief, and memory in altered form. It does not restore innocence. The first time Chara resets, it is because she lost and needs to try again. The reset does not undo the fact of loss.

The soul property most closely tied to reset access is Determination. The manuscript confirms that Chara's Determination level is high enough to retain memory through events that lower-DT individuals cannot retain. This is the same property [[lore/lab-c7|Lab C7]] classified as **THE RED GENOCIDE**.

## The Reset Anchor

Raiden builds a reset contingency specifically for Chara — described in the text as a **dim yellow light** intended for _"the moment when she would be alone and broken."_

This anchor is not a general safety net. It is built for one person, for a specific kind of last-resort survival. It carries Raiden's intent: that she should have one more option when everything else is gone.

## Timeline 284

The manuscript references Timeline 284 in connection with HChara. Chara retains full memory of Timeline 284 events that [[characters/hchara|HChara]] experienced. The exact relationship between Timeline 284 and the main story sequence — whether it is a prior loop, a divergent branch, or something else — is covered in [[timeline/alternate-timelines|Alternate Timelines]].

## The SAVE Star

The flashback material references a **SAVE Star** found in the Library data room, taken by Chara during her escape from [[lore/lab-c7|Lab C7]]. The exact mechanics of the SAVE Star — whether it creates a new anchor, restores a prior one, or serves a different function — are not fully documented in reviewed sources.

> **Unresolved:** The relationship between the SAVE Star and the reset anchor Raiden built is not confirmed.

## Permanent Removal: Arc 2.3

The reset architecture is taken away permanently in Arc 2.3. Axel confirms that [[lore/soul-fusion|soul fusion]] with a Determination-type soul severs timeline anchors irreversibly:

> _"No resets, no reloads. This is your only life now."_

After this moment, consequences stop being provisional. The manuscript treats this as a permanent narrowing of Chara's options — the same Determination that made resets possible is now locked into a single forward-running timeline.

## Significance

The removal of the reset option is the point at which _Fractured Hope_ commits to its stakes. Everything before Arc 2.3 carries a theoretical reversibility; everything after is final. The story's violence, deaths, and choices after this point land differently because of it.
